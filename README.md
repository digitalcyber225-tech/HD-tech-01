# StockPro - Application de Gestion de Stock

## Fichiers inclus
- `index.html` — Application principale (tout-en-un)
- `manifest.json` — Fichier PWA pour installation mobile
- `sw.js` — Service Worker (mode hors ligne)

## Comment déployer et partager

### Option 1 : Netlify (GRATUIT - le plus simple)
1. Allez sur https://netlify.com et créez un compte gratuit
2. Glissez-déposez le dossier `stock-app` dans la zone "Déposer votre site"
3. Netlify vous donne un lien comme : `https://stockpro-abc123.netlify.app`
4. Partagez ce lien à vos clients !

### Option 2 : GitHub Pages (GRATUIT)
1. Créez un compte sur https://github.com
2. Créez un nouveau dépôt public
3. Uploadez les 3 fichiers
4. Allez dans Paramètres → Pages → Activer GitHub Pages
5. Lien automatique : `https://votre-nom.github.io/stockpro`

### Option 3 : Vercel (GRATUIT)
1. https://vercel.com → Créer compte
2. Importer les fichiers → Déployer
3. Lien partageable immédiat

## Comment installer sur téléphone (pour vos clients)

### Android (Chrome) :
1. Ouvrir le lien dans Chrome
2. Menu (3 points) → "Ajouter à l'écran d'accueil"
3. Ou cliquer la bannière d'installation qui apparaît

### iPhone (Safari) :
1. Ouvrir le lien dans Safari
2. Bouton Partager (carré avec flèche)
3. "Sur l'écran d'accueil"
